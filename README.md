#  Access Management Maturity Model
**Versão:** 1.0  
**Base normativa:** ISO/IEC 27001:2022 + ISO 27701  

## Objetivo
Modelo independente de avaliação de maturidade em Gestão de Acessos, transformando controles normativos em checklist interativo.  
Permite testar controles de acesso e identidade de forma prática, sem depender de dados reais de empresa ou cliente.

## Controles Avaliados
- 5.15 – Access Control
- 5.16 – Identity Management
- 5.17 – Authentication Information
- 5.18 – Access Rights
- 8.2 – Privileged Access Rights
- 8.3 – Information Access Restriction
- 8.5 – Secure Authentication

## Modelo de Pesos
- Peso 5 – Estrutural Crítico  
- Peso 4 – Governança Forte  
- Peso 3 – Controle Operacional Importante  
- Peso 2 – Controle Complementar
